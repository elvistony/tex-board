# tex-board
#### Under Development
A Text Supported Board that works Peer-to-Peer

### Features
- Simple Presentation/Smart Board
- Text Selectable Sharing
- Peer-side rendering to reduce Data usage

### Current Features
- Make New Text Boxes
- Move Text Boxes
- Resize Text Boxes
- Dynamic Font Size to Fit Text
